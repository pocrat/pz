  • Event apps in the early 2010s

  • Description

    Early 2010s (to me, 2012 - 2014) was what I call the “80s of mobile UI design”. It was a time when there were 4 major mobile platforms - Blackberry and Windows(Nokia) phones were trying hard to regain their market shares. The mobile design languages / systems were in their infancy. Web frameworks and UI libraries weren't so popular.

    What do you do when the clients ask to build their seasonal event apps on all 4 platforms? You give plenty of personalities to each app! As you will see, my first job as a UI designer was an incredibly fun gig. These apps' main functionalities were similar due to the fact they shared the same core product. Designing their UI skin and testing them on various devices was incredibly fun.
