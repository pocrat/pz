  • GoXLR mixer companion app UI

    Hardware configuration & maintenance made easy thanks to its best friend.

  • Description

    GoXLR is an audio mixer made for online broadcasters. My job was to create an interface solution for the GoXLR companion software on Windows 10 OS. Like most products on the market, it must possess both function and form - the software needed to be functional, beautiful and easy to use. The companion software helps users expedite customized setup, manage audio samples, device updates & maintenance. We wanted the users to spend less time doing configuration and more time creating content with the hardware.
